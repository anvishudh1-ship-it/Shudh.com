Manhole Analysis Report

Introduction

This report presents an analysis of manhole data using two provided datasets: a static dataset containing manhole attributes and a records dataset with operational information. The analysis focuses on understanding the relationship between manhole characteristics and operational metrics.

Summary of Data

Static Manhole Data

The static dataset contains information about 50 manholes, including:

  • Manhole ID: Unique identifier for each manhole.
  • Physical Attributes: Diameter, depth, type of manhole, type of pipeline.
  • Operational History: Last operation date, history of clogging incidents, condition.
  • Environmental Factors: Nearby land use, population density.
  • Flow Data: Flow in and out, water level.

Operational Records

The records dataset contains a log of operations, including:

  • Operation Time: The duration of each operation in minutes.
  • Gas Data: Readings for CO, CH4, and H2S.
  • Timestamp: The date and time of the operation.
